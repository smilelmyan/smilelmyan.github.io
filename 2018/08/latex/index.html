latex及字体安装

  2018-08-12   Ubuntu桌面应用   latex  Comments Word Count: 1875words Read Count: 4minutes

安装TexLive

archlinux的官方仓库，有textlive-most和texlive-lang两个组包，不想折腾每个组件都是做什么的，most全部安装。

$ sudo pacman -S texlive-most
$ sudo pacman -S texlive-langchinese texlive-langextra

其中texlive-lang里面只安装了和中文相关的。

安装字体

可以从windows里复制字体，把常用的中英文字体复制到创建msfont文件夹里

$ sudo  mkdir /usr/share/fonts/msfont/
$ sudo cp $WIN_FONTS /usr/share/fonts/msfont/

更新font cache使Linux系统识别新加入的字体。

$ cd /usr/share/fonts/msfont/
$ sudo mkfontscale
$ sudo  mkfontdir
$ sudo fc-cache -vf

使用

写论文的时候，新建paper.tex，使用xelatex引擎编译。用过vim写过latex，有好多更容易用的前端软件，vim的latex插件。

\documentclass[12pt,a4paper,openany]{book}
% utf-8编码，xelatex编译引擎
  \input{Head.tex}
%
\begin{document}
%--- 首页
\begin{titlepage}
\begin{center}
\includegraphics[width=6.5cm]{logolzu.pdf} 
\vspace{4em}

{\xiaochu \kaishu 毕~~业~~论~~文}

\sihao

{( 本~科~生 )}
